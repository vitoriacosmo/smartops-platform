# SmartOps Platform

Plataforma de monitoramento inteligente de pedidos com microsserviços e IA Generativa.

## Serviços
- **order-service** (Kotlin + Spring Boot) — API REST de pedidos
- **analytics-service** (Python + FastAPI) — análise de eventos
- **ai-insights-service** (Python) — insights com IA Generativa
- **notification-service** (Kotlin) — notificações

## Tecnologias
Kotlin · Python · Spring Boot · FastAPI · Kafka · PostgreSQL · Docker · GitHub Actions · Grafana

## Como rodar localmente
```bash
docker-compose up -d
```
